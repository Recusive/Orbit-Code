Minimal interactive debug client for the Codex app server (protocol v2).

### What this folder does

A tiny command-line client that spawns `codex app-server` as a subprocess and communicates with it over stdin/stdout using JSON-RPC. Prints all server notifications and responses, lets users type turns interactively, and supports thread management (new, resume, switch). Primarily used for debugging and testing the app server protocol.

**Note:** This code is mainly generated by Codex and is not intended for production use.

### Where it plugs in

- Spawns `codex app-server` as a child process
- Communicates via the app server protocol v2 (JSON-RPC over stdio)
- Uses `codex-app-server-protocol` for protocol types

### Imports from

- `codex-app-server-protocol` -- `AskForApproval` and protocol types
- `clap` -- CLI argument parsing
- `serde` / `serde_json` -- JSON-RPC message serialization
