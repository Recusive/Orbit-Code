# codex-rs/skills/src/assets/samples/skill-creator/

Meta-skill for creating new Codex skills.

## What this folder does

A built-in skill that helps users create new Codex skills. Includes initialization scripts, validation tools, and reference documentation about the skill YAML format.

## Key files

- `SKILL.md` -- skill metadata and description.
- `license.txt` -- license file.
- `agents/openai.yaml` -- agent configuration.
- `assets/` -- skill icons.
- `references/` -- reference docs (openai_yaml.md with YAML format specification).
- `scripts/` -- Python scripts for skill creation and validation.
