# codex-rs/skills/src/assets/samples/skill-installer/

Skill for installing Codex skills from GitHub repositories.

## What this folder does

A built-in skill that helps users discover and install skills from GitHub. Includes scripts for listing available skills and installing them.

## Key files

- `SKILL.md` -- skill metadata and description.
- `LICENSE.txt` -- license file.
- `agents/openai.yaml` -- agent configuration.
- `assets/` -- skill icons.
- `scripts/` -- Python scripts for skill discovery and installation.
