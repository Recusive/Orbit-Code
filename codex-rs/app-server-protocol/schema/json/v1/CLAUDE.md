# app-server-protocol/schema/json/v1

## Purpose

Auto-generated JSON Schema files for v1 protocol types. The v1 protocol is a legacy/compatibility layer; only a small set of types are generated here.

## Key Files

| File | Role |
|------|------|
| `InitializeParams.json` | JSON Schema for the v1 initialize request parameters |
| `InitializeResponse.json` | JSON Schema for the v1 initialize response |

## Regeneration

Generated by `cargo run --bin write-schema-fixtures` or `just write-app-server-schema`. Do not edit manually.

## What It Plugs Into

- Part of the broader JSON Schema fixture set validated by `tests/schema_fixtures.rs`.
- Generated by `src/export.rs` which filters v1 types via `JSON_V1_ALLOWLIST`.

## Exports To

- External consumers needing v1 protocol JSON Schema definitions.
