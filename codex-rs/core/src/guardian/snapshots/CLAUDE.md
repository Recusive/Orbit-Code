# codex-rs/core/src/guardian/snapshots/

Insta snapshot files for guardian review tests.

## What this folder does

Contains `.snap` files used by the `insta` snapshot testing library. These snapshots capture the expected layout/structure of guardian review requests, ensuring that changes to the guardian prompt or request formatting are intentional and reviewed.

## Key files

| File | Purpose |
|------|---------|
| `codex_core__guardian__tests__guardian_review_request_layout.snap` | Snapshot of a standard guardian review request layout |
| `codex_core__guardian__tests__guardian_followup_review_request_layout.snap` | Snapshot of a follow-up guardian review request layout |

## Where it plugs into

- **Generated by**: `guardian/tests.rs` using `insta::assert_snapshot!()` macros
- **Updated via**: `cargo insta review` when the format intentionally changes
