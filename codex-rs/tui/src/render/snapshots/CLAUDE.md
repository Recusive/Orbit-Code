# codex-rs/tui/src/render/snapshots/

Insta test snapshots for the rendering engine.

## What this folder does

Contains `.snap` files generated by the `insta` crate during test runs. These snapshots capture the expected output of rendering tests, such as ANSI color family foreground palette rendering for the syntax highlighting engine.

## What it plugs into

- **../highlight.rs**: Tests in the highlight module use `insta::assert_snapshot!()` to verify rendered output against these snapshot files.
- Snapshot files follow the naming pattern `codex_tui__render__highlight__tests__*.snap`.
