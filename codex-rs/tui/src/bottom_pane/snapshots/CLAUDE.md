# codex-rs/tui/src/bottom_pane/snapshots/

Insta test snapshots for bottom pane component tests.

## What this folder does

Contains `.snap` files generated by the `insta` crate during test runs. These snapshots capture the expected rendered output of various bottom pane components including the approval overlay (exec and patch approval prompts, additional permissions, cross-thread prompts), app link views (enable/install suggestions), and other interactive overlays.

## What it plugs into

- Tests in `bottom_pane/approval_overlay.rs`, `bottom_pane/app_link_view.rs`, and other bottom pane modules use `insta::assert_snapshot!()` to verify rendered output.
- Snapshot files follow the naming pattern `codex_tui__bottom_pane__<module>__tests__*.snap`.
