# codex-rs/tui_app_server/src/render/snapshots/

Insta snapshot files for render module tests.

## What this folder does

Contains `.snap` files generated by the `insta` snapshot testing crate. Each file captures the expected output of rendering functions (syntax highlighting, line utilities, renderable layout) under specific test conditions.

## What it plugs into

- **../**: Test functions in render module files use `insta::assert_snapshot!()` to compare output against these golden files.
- **insta** (dev-dependency): Manages snapshot creation, diffing, and review.

## Key files

All files follow the naming convention `codex_tui_app_server__render__<submodule>__tests__<test_name>.snap`.
