# codex-rs/tui_app_server/src/bottom_pane/snapshots/

Insta snapshot files for bottom pane tests.

## What this folder does

Contains `.snap` files generated by the `insta` snapshot testing crate. Each file captures the expected rendered output of bottom pane components (approval overlays, composer states, selection views, status lines, etc.) under specific test conditions.

## What it plugs into

- **../**: Test functions in the bottom pane module files use `insta::assert_snapshot!()` to compare rendered output against these golden files.
- **insta** (dev-dependency): Manages snapshot creation, diffing, and review.

## Key files

All files follow the naming convention `codex_tui_app_server__bottom_pane__<submodule>__tests__<test_name>.snap`.
