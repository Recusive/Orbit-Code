# codex-rs/tui_app_server/src/status/snapshots/

Insta snapshot files for status module tests.

## What this folder does

Contains `.snap` files generated by the `insta` snapshot testing crate. Each file captures the expected rendered output of status formatting tests (rate limit displays, status cards, account info, token summaries).

## What it plugs into

- **../tests.rs**: Test functions in the status test module use `insta::assert_snapshot!()` to compare output against these golden files.
- **insta** (dev-dependency): Manages snapshot creation, diffing, and review.

## Key files

All files follow the naming convention `codex_tui_app_server__status__tests__<test_name>.snap`.
